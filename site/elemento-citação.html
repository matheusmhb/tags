Elementos de Citação

Em HTML, existem elementos específicos para indicar citações e referências dentro do conteúdo textual de uma página da web. Esses elementos ajudam a fornecer contexto e credibilidade ao conteúdo e facilitam a compreensão do leitor sobre as fontes das informações apresentadas.

<abbr></abrr> -Tag HTML define uma abreviação ou sigla dentro do texto. O texto dentro de uma tag é geralmente exibido com uma linha pontilhada sob ele, para indicar que é uma abreviação.

Ex:

Estamos aprendendo HTML.


<u></u> -Tag HTML define as informações de contato do autor/proprietário de um documento ou artigo.

As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, nome de usuário em redes sociais, etc.

O texto dentro da tag geralmente é renderizado em itálico.

Ex:

Rua do Exemplo , 123 , centro, cidade



<cite></cite> -Tag HTML define o título de uma obra criativa (por exemplo, um livro, um poema, uma música, um filme,etc.).

O texto no elemento geralmente é exibido em itálico .

Ex:

- Benjamin Franklin


<Q></Q> -A tag HTML define uma citação curta, dentro do texto principal da página. O texto dentro de uma tag Ex; geralmente é envolvido por aspas, para indicar que é uma citação.
