Listas e tabelas

As lista em HTML são usadaas para agrupar e exibir itens relacionados. Por exemplo, você pode criar uma lista com um passo a passo para executar alguma tarefa, ou uma lista de ingredientes para fazer um bolo. Hoje temos no HTML as listas ordenadas e as não ordenadas

Lista Não Ordenadas

  • Aroz
  • Feijão
  • Macarrão
  • Sal
  • Açucar

Uma lista não ordenada é criada usando a tag <ul></ul> que representa uma lista de itens não ordenados. Cada item da lista é representado pela tag <li></li>.


Listas Ordenadas

  1. Aroz
  2. Feijão
  3. Macarrão
  4. Sal
  5. Açucar

Uma lista ordenada é criada usando a tag <ol></ol>, que representa uma lista de itens ordenados. Cada item da lista é representado pela tag <li></li>.


Tabelas

A tag utilizada para criar uma tabela HTML é a tag <table></table>.Dentro dela, incluímos todos os elementos que compõem nossa tabela, ou seja, as células da tabela. Assim sendo, os elementos de uma tabela consistem em outras tags que poderão ser utilizadas.

As tags que vão formar a estrutura básica de uma tabela em HTML são as tags <tr></tr> e <td></td>. A tag <tr> representa uma linha e a tag <td> representa uma célula.
